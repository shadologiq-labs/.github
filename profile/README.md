ShadoLogiq Labs builds tactical tools — browser extensions, userscripts, and embeddable widgets — that sit between rigid enterprise platforms and the people who use them every day. When a vendor's default interface respects neither your time nor your cognitive load, we fix it at runtime.

Everything we ship is open source, runs entirely on your machine, and has zero monetization or telemetry. No accounts, no servers, no central approval queue.

---

## What we build

### 🚀 [Gapps Embed](https://github.com/shadologiq-labs/gapps-embed) — `v1.0.0`

A lightweight toolkit for embedding a Google apps grid on any site. Three presentations share one data core, one sprite, and zero build steps:

- **Launcher** — plain icon grid, identical to Google's own ✅ shipping now
- **Spotlight** — card grid with descriptions and "Learn more" links 🛠 v1.1
- **Menu** — waffle button + popover panel 🛠 v1.2

Drop in one `<iframe>`, theme it via query string or `postMessage`, and rewrite URLs for any Workspace domain.

[Live demo](https://shadologiq.com/gapps-embed/) · [Repo](https://github.com/shadologiq-labs/gapps-embed) · [Docs](https://github.com/shadologiq-labs/gapps-embed#readme)

### 📌 [SNapp Extension](https://github.com/shadologiq-labs/snapp-ext) — *launching soon*

A right-click upgrade for ServiceNow. Copy richly-formatted record links with metadata, inject standardized change templates, and auto-extract URLs and email contacts from form fields.

Manifest V3 · Chrome, Edge, Firefox · all processing in-page · no network calls

[Repo](https://github.com/shadologiq-labs/snapp-ext) · store listings coming
